# AI_Kanban_Board

1. Three columns to start: To Do, In Progress, Done — with the ability to add more

2. A card count on each column header

3. Each card has a title, a priority (Low, Medium, or High), and an assignee name

4. Cards can be moved between columns

5. Cards can be deleted

6. A priority filter that lets you choose which priority level to show across all columns

7. A search bar that filters cards across the entire board by title or assignee
